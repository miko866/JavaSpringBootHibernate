# Client Manager Sytsem in Vue JS and Java Spring Boot Hibernate with Axios

## Instalation

### Database
At First create MySQL DB from:
```
db.sql
```

### Client
In root folder:
```
npm install
```

### Server
Simple import Maven changes.
In resources:
```
application.properties
```
change DB settings!

## RUN

### Client 
In root folder
```
npm run serve
```
Server start at 
```
http://localhost:4300/
```

### Server 
Run
```
SimpleManagerSystemApplication.java
```
